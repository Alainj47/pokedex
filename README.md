# Ejecutar en dev

1. Clonar repo
2. Ejecutar
``` 
yarn isntall
``` 
3. Instalar Nest CLI
``` 
npm i -g @nestjs/cli
``` 

4. Levantar DB
``` 
docker-compose up -d
``` 

## Stack usado
* MongoDb
* Nest
